# My App

Это веб-приложение, созданное с использованием React, TypeScript и Vite.

## Стек технологий

- **Фреймворк:** React
- **Язык:** TypeScript
- **Сборщик:** Vite
- **Стили:** CSS Modules
- **Маршрутизация:** React Router
- **HTTP-клиент:** Axios
- **Управление состоянием:** Zustand & React Query
- **Формы:** React Hook Form & Zod
- **Тестирование:** Jest & React Testing Library

## Локальный запуск

1. **Установите зависимости:**

   ```bash
   npm install
   ```

2. **Запустите development-сервер:**

   ```bash
   npm run dev
   ```

   Приложение будет доступно по адресу [http://localhost:5173](http://localhost:5173).

## Запуск в Docker

1. **Соберите и запустите контейнер:**

   ```bash
   docker-compose up -d
   ```

   Приложение будет доступно по адресу [http://localhost:5173](http://localhost:5173).

2. **Остановите контейнер:**

   ```bash
   docker-compose down
   ```
